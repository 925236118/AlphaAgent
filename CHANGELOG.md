# Changelog

All notable changes to this project will be documented in this file.

## [0.4.0] - 2026-02-12

### ✨ New Features (新增功能)
- **文件变更追踪**:
  - 显示修改的文件列表，支持查看文件变更。
  - 添加文件对比窗口，支持差异对比算法和同步滚动。
  - 支持保留更改或撤回到源版本。
- **技能功能**:
  - 添加技能系统，支持加载和管理技能。
  - 添加技能管理面板。
  - 添加默认技能和插件欢迎语。
- **Tools**:
  - 添加为场景添加节点工具，在当前编辑的场景中添加节点。
  - 添加获取当前场景目录树工具，读取当前打开场景的节点树及节点属性。
  - 添加创建脚本工具，创建脚本文件。
  - 添加创建场景或资源工具，创建场景或资源。
  - 添加全局搜索工具，全文搜索项目内容。
  - 添加资源内容读取工具，读取资源文件内容结构。
  - 添加读取脚本大纲工具，读取脚本文件大纲。

### 🐛 Bug Fixes (问题修复)
- 修复记忆面板中切换 Godot 侧边栏时记忆数量变化错误的问题。
- 修复写文件工具会创建文件夹而非文件的问题。
- 修复规划列表完成后，再次发送对话未清空并关闭规划列表的问题。
- 修复拖动内容到对话框内后失去光标和焦点的问题。
- 修复 DeepSeek 模型下出现的错误转义字符问题。

### 🔧 Improvements (优化改进)
- 重构工具调用脚本，优化工具调用流程和代码逻辑。
- 添加全局配置文件统一管理（memory_file 属性）。
- 优化输入框和设置页面统一样式。

## [0.3.0] - 2025-01-09

### ✨ New Features (新增功能)
- **UI**:
  - 重构整个Agent的各个组件和功能页面的样式。
  - 在发送键为 Enter 时，按 Ctrl+Enter 应换行。
  - 优化各类按钮样式。
- **模型管理**:
  - 添加供应商配置功能，可调用各种 OpenAI 风格的供应商和 Ollama。([#1](https://github.com/925236118/AlphaAgent/pull/1))
  - 添加多个默认供应商及其模型，优化供应商管理功能。
  - 添加模型管理界面，添加检测供应商是否可用功能。
  - 添加点击获取模型列表功能。
  - 重构调用模型节点为随时创建，减少用户 secret_key 泄露风险。
- **角色功能**:
  - 开发角色提示词功能。添加三个默认角色以及相关提示词和工具配置。
- **对话功能**:
  - 添加停止和错误展示功能。
  - 添加结束提示，并添加重新发送功能。
  - 添加复制按钮。
  - 删除Ask和Agent模式。
- **任务规划**:
  - 添加规划任务功能及相关 UI。
  - 优化任务规划 UI，新建对话时清除任务规划列表。
- **Tools**:
  - 添加获取 TileSet 信息工具，封装获取节点函数。
  - 添加命令行调用工具，默认将在项目目录下执行。
  - 添加 update_scene_node_property 工具。
  - 添加工具将资源文件挂载到指定属性上（允许嵌套资源）。
  - 添加工具将资源引用给指定节点的指定属性上。
- **历史记录**:
  - 优化历史记录功能。
  - 修改历史记录保存位置。
  - 修改历史记录和标题按钮为 Window 弹窗。
- **记忆功能**
  - 优化记忆功能页面。

### 🐛 Bug Fixes (问题修复)
- 修复多次点击历史记录会创建多次问题。
- 修复输入框占位词错误问题。
- 修复下拉框遮挡输入的问题。
- 修复模型管理中显示的支持标签错误问题。
- 删除项目中存在的安全密钥。
- 修复所有的按钮的Tooltip文案([#2](https://github.com/925236118/AlphaAgent/issues/2))

### 🔧 Improvements (优化改进)
- 移除历史记录文件并调整设置路径初始化逻辑。
- 插件启动单例模式完善。
- 抽象出 AgentModelUtils 类作为所有模型通用工具集。
- 更新部分工具描述和参数限制。
- 精简提示词。
- 修改帮助内容
- 为全局设置文件添加版本号，防止多版本之间互相影响
- 添加爱赞助链接
