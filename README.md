# Godot AI Agent 项目

## 项目简介

**Godot AI Agent** 是一个专为Godot引擎设计的智能代理插件，旨在帮助游戏开发者快速、高效地开发游戏项目。通过集成先进的AI技术，本插件提供了智能对话、代码生成、场景管理等功能，让开发者能够专注于游戏创意和核心逻辑的实现。

## 主要特性

- **智能开发助手**: 提供AI驱动的代码生成和优化建议
- **快速原型制作**: 通过对话式交互快速创建游戏原型
- **场景管理**: 智能场景构建和资源管理
- **工具集成**: 集成多种开发工具，提升开发效率
- **流式聊天界面**: 提供流畅的AI对话体验

## 项目目录结构

**根目录文件:**
- `project.godot` - 项目配置文件
- `README.md` - 项目说明文档
- `icon.svg` - 项目图标
- `test_editor_script.gd` - 测试脚本
- `.gitignore` - Git忽略文件
- `.gitattributes` - Git属性文件

**插件目录 (res://addons/agent/):**
- `plugin.cfg` - 插件配置文件
- `agent.gd` - 主代理脚本
- `scripts/deepseek_chat_stream.gd` - DeepSeek聊天流脚本
- `tools/tools.gd` - 工具脚本
- `ui/main_panel.tscn` - 主面板场景
- `ui/main_panel.gd` - 主面板脚本
- `ui/message_item.tscn` - 消息项场景
- `ui/message_item.gd` - 消息项脚本
